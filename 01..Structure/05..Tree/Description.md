# 트리 (Tree) 구조

**트리 구조**(나무구조)란 그래프의 일종으로, 여러 노드가 한 노드를 가리킬 수 없는 구조이다. 간단하게는 회로가 없고, 서로 다른 두 노드를 잇는 길이 하나뿐인 그래프를 트리라고 부른다.

주로 이진 트리 (Binary Tree) 형태 구조로 많이 사용되고, **탐색 알고리즘** 구현에 많이 사용된다.
